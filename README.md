# Portfolio website
Personal Portfolio website using Next.JS, tailwind, typescript and sentry.
### Libraries used:
- Aceternity.ui https://ui.aceternity.com/
- Framer Motion https://www.framer.com/motion/
- shadcn/ui https://ui.shadcn.com/
### To run dev version
```
npm run dev
```
